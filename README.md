## wireguard-install
WireGuard [road warrior](http://en.wikipedia.org/wiki/Road_warrior_%28computing%29) installer for Ubuntu, Debian, AlmaLinux, Rocky Linux, CentOS and Fedora.

This script will let you set up your own VPN server in no more than a minute, even if you haven't used WireGuard before. It has been designed to be as unobtrusive and universal as possible. For correct installation, yggdrasil must be installed on the server.

### peculiarity

provides access to the yggdrasil network by distributing ip addresses to clients from the subnet addresses of the main ip yggdrasil
